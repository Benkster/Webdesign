=======================
CMS:
=======================



x 	CMS = Content Management System
x 	Different types:
	CMS
	ECM (Enterprise Content Management)
	WCM (Web Content Management)

x 	CMS:
	x 	precursor to ECM
	x 	works best with structured content such as documents & DB records
x 	ECM:
	x 	manages structured and unstructured content
	x 	includes software, strategies and security to effectively manage content
	x 	focuses on content production
	x 	stores digital files in nested folders
x 	WCM:
	x 	similar to ECM but for web content
	x 	manages multiple types of content
	x 	externally oriented
	x 	stores data in DB

x 	Allowes users to create, edit and publish content in an easy way
x 	GUI provides tools -> no need to write code from scratch
x	2 Components:
	CMA (Content Management Application)
	x	GUI, that allows to design, create, modify, remove content from website
	x 	Knowledge in HTML, CSS and JS is often not needed

	CDA (Content Delivery Application)
	x 	provides back-end service -> code behind that runs the system
	x 	actual management happens here

x 	Popular features:
	x 	search engine
	x 	permission system and security
	x 	templates
	x 	file uploads

x 	Benefits of CMS:
	x 	Easy to use due to GUI
	x 	Easy to search information
	x 	Easy to manage content
	x 	Accessible from anywhere
	x 	Manage permissions
	x 	Instant content updates
	x 	Fast managing

x 	Examples:
	x 	Joomia
	x 	WordPress
	x 	Wix
