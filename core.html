Core elements

HTML

HTML stands for Hyper Text Markup Language and is the standard markup language for creating web pages.

It describes the structure of a web page and consists of a series of elements. Those elements tell the browser which content to display and labels pieces of the content such as a heading, a paragraph or a link.

CSS

CSS is short for Cascading Style Sheets and it defines, how elements are beeing displayed on the screen, on paper or on other media types.

The color, font and size of a text, spacing between elements, the positioning of elements, background settings - like images or colors - and much more are specified via CSS.
The CSS can be specified in an external style-sheet, an internal style-sheet or an inline-style.

JS

JS stands for Java Script and it makes HTML pages more dynamic and interactive.
It can be specified via external js-file or internal js-script.

Common uses for JS are:
  • element manipulation
  • form validation
  • dynamic content changes
