Progressive Web Apps

What are PWAs

PWAs are websites, that have the capability to be installed, run offline and use native device APIs like the accelerometer or cameras.

They use emerging web browser APIs and features along with traditional progressive enhancement strategy to bring a native app-like user experience to cross-platform web applications.

Progressive Web Apps should be discoverable, installable, linkable, network independent, progressive, re-engageable, responsive, and safe.

Requirements

  • A PWA needs to use the https protocol to ensure security
  • It needs a service worker to allow for offline capabilities
  • It requires a manifest.json file to declare how the app should be called, what icons should be used and what options it has.

Usecases

  • Building an App once and deploying it on all platforms with a single codebase
  • Send push notifications on websites
  • Take photos directly on your websites like on https://selbsttest.tirol/
  • Allows tools like google docs to be used without an active internet connection
