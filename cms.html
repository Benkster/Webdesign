CMS

There are three different types:

CMS Stands for Content Management System

  • It can be seen as the precursor to the ECM, since it is based on the CMS.
  • CMS works best with structured content
    Eg. documents or database records
ECM Stands for Enterprise Content Management

  • Can manage structured and unstructured content
  • Includes software, strategies and security to effectively manage content
  • ECM focuses on content production
  • Digital files are stored in nested folders
WCM Stands for Web Content Management

  • It is similar to the ECM but specifically for web content
  • Manages multiple types of content
    Eg. plain text, code, pictures, ...
  • Externally oriented
  • Stores data in a database

General info

CMS, ECM and WCM allow users to easily create, edit and publish content. Thanks to a GUI that provides several tools, there is no need to write the code from scratch.


CMA (Content Management Application)
  • GUI, that allows to design, create, modify and remove content from the website
  • In many cases, knowledge in HTML, CSS and JS is not required


CDA (Content Delivery Application)
  • It provides the back-end service
  • The actual management and all processes happen here

Popular features

  • Search engines
  • Permission systems and security
  • Templates
  • File uploads

Benefits of a CMS

  • It is easy to use due to a GUI with great overview
  • Searching information is very easy and takes only a short time
  • Managing content is simple and fast
  • It is accessible from anywhere
  • Content is beeing updated instantly
